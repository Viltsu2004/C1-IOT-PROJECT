# C1-IOT-PROJECT
C-language project, second year

Project is from passengers airplane seat palces. 

# How program works
- 26 rows of seats
- Each row contains seats A-F
    -> Seats A and F are by window, B and E are in the middle and C and D are by the aisle.
- User eneter name, row number and seat place.
- User can print seat-map of the airplane
    -> User sees from seat map which seats are already booked and which are free
- Reservation is added in to the reservation file
- Whole information of file is printed sorted by last and firstname

Project is done by using c-language. And it's part of C-course.
